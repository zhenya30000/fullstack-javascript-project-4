## Installation

To install clone the project in the desired folder
```sh
git clone git@github.com:zhenya30000/fullstack-javascript-project-4.git
```

And use npm ci
```sh
npm ci
```
Or Makefile command 
```sh
make install
```
To use CLI, link the project in the system
```sh
sudo npm link
```
or 
```sh
make link
```
